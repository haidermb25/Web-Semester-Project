MedRegis – Doctor Registration Management System

This project is a web-based Doctor Registration Management System built as a semester project for managing doctor registrations, profiles, and related administrative data.

📖 About

A semester project that provides a digital system for managing doctor registrations, including adding, updating, and organizing doctor records efficiently.

🚀 Features
Doctor registration system
Profile management
Record updates and deletion
Admin panel functionality
Simple web-based interface
CRUD operations
